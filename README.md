# Qa::Capybara

Simple Capybara Console.

## Installation

Add this line to your application's Gemfile:

    $ git clone git@github.com:kowal/qa-capybara.git
    $ bundle

## Usage

	$ ruby bin/qac # opens Pry console with capybara waiting on google.com
