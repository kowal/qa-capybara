# Qa::Capybara

Simple Capybara Console.

Work in progress..

## Installation

Add this line to your application's Gemfile:

    $ git clone git@github.com:kowal/qa-capybara.git
    $ bundle

## Usage

	$ ruby bin/qac http://www.google.com
	# opens Pry console with capybara waiting on google.com
